# Weekly Macro Market Brief

**An automated AI-powered market intelligence system.**

This Python application pulls live business and geopolitical news from NewsAPI, analyzes it with OpenAI using strict JSON schemas and advanced prompt engineering, and instantly generates clean, professional weekly macro briefings for U.S. equities.

### What It Delivers
- DJIA and S&P 500 1-day & 5-day bias with confidence levels
- Key drivers, counterarguments, and “what would change the view”
- Multi-audience summaries (Explain Like I’m 5 → Professional)
- Overall sentiment score and cautious long-position/re-entry guidance
- Slick & professional looking HTML page + downloadable PDF

### Tech Stack
- **Python** (core pipeline)
- **NewsAPI** (real-time news fetch)
- **OpenAI** (structured outputs + detailed prompt engineering)
- HTML/CSS templating + HTML-to-PDF conversion

Fully automated, container-ready, and deployed on GitHub Pages.
